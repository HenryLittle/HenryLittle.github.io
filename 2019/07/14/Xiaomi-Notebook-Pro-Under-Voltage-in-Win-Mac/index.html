Xiaomi Notebook Pro Under Voltage in MacOS

2019-07-14

preface

Using XTU or VoltageShift to undervolt and adjust powerlimit to get better thermal & performace. Also improves battery life in MacOS.

requirement

  • Mi Notebook Pro MX150 (tested on i7 16G model)

  • BIOS version 603

  • UnderVolt Patch

    MacOS

  • patch the bios in windows first

  • use XTU to find the proper voltage offset (My values -135mv for CPU)

  • change the onwer of the kext file

  • use votageshift to set powerlimit and offset

  • reboot (if things goes wrong use F3 to boot into the Recovery OS and run csrutil)
